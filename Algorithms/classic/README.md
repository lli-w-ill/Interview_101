## 1. Stack memory vs Heap memory

|  Stack Memory  |  Heap Memory  |
| -------------- | ------------- |
| small size     | large size    |
| fast access    | slow access   |
| stores function calls and local variables | stores objects |
| no framentation | may become fragmented |
